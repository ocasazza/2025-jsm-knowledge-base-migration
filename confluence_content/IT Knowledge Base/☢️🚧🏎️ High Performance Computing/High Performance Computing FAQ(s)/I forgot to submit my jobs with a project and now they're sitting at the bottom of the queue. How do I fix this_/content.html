You can alter your pending jobs with the command:

qalter -P

If you want to switch all of your jobs to the same project, you can run the following command, substituting your username and project name as appropriate. Note this is all one line:

for i in qstat -s p -u <username>|tail -n +3 | awk '{print $1}' ; do qalter -P $i; done

true User Story

As a researcher, I need to know how to update the project association for my already submitted jobs so I can ensure they receive the appropriate priority in the queue.
