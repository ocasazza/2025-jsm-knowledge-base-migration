There are two queues for running CPU jobs, cpu.q and cpu_short.q. Jobs on cpu_short.q have 30 minutes hard wall clock limit. There are two ways to submit jobs to the CPU queues.

  • With Schrodinger host file
  • Use "bolt_cpu" for running jobs on cpu.q
  • Use "bolt_cpu_short" for running jobs on cpu_short.q.
  • Without using Schrodinger host file
  • Use "-l cpuonly -pe one_node " for running jobs on cpu.q
  • Use "-l short -l cpuonly -pe one_node -l h_rt=00:30:00" for running jobs on cpu_short.q.
  • Submitting jobs with Schrodinger host file is strongly recommended.

true User Story

As a researcher, I need to know how to submit jobs to CPU queues so I can run calculations that don't require GPU resources.
