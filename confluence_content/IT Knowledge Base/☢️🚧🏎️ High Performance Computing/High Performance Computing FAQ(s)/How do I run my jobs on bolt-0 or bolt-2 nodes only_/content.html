Bolt-0 nodes have GTX1080 GPU cards installed; bolt-2 nodes have GTX1080Ti GPU cards. If for any reason, you want to run all your jobs on the nodes with same type of GPU cards, you can add following additional complex resource to QARGS to your job submission script.

GPU card type QARGS
GTX 1080 -l gtx1080
GTX 1080Ti -l gtx1080ti

Note: Requesting a particular type of GPU cards (nodes) will decrease the resource pool for the jobs and therefore it takes longer for the jobs to be dispatched. If a job runs fine with mixed GPU card types, it's recommended not requesting specific type GPU cards.

true User Story

As a researcher, I need to know how to target specific GPU hardware for my jobs so I can ensure consistent performance or take advantage of specific hardware features.
