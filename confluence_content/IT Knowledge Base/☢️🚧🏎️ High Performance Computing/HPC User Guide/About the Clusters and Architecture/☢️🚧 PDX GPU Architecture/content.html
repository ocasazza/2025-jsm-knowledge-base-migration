Summary

The pdxgpu cluster consists of 34 GPU cards (distributed across 9 GPU compute nodes with varying configurations), 2 non-GPU compute nodes (for master jobs), 1 interactive submission node, and 1 master node. It shares storage with bob, runs the CentOS 6.5 operating system and SGE 2011.11p1 queuing system.
