If you have previously manually connected to/configured a network drive on our VNX storage (pdx-stor-vnx or nyc-stor-vnx) on your own machine, you will need to make changes to the connection settings because we've migrated to new Qumulo storage.

macOS

If you use the macOS “Connect to Server…” menu, make sure you use the path:

smb://schrodinger.com/shares/{ nyc | pdx }/sharename

Example
bash

In NYC, no changes are needed when connecting to the DDAG share, but changes needed when connecting in PDX.

Make sure you use smb:// , and not cifs:// when specifying the address.

Windows

From Windows , make sure you use the path:

\\schrodinger.com\shares\{ nyc | pdx }\sharename

bash

In NYC, no changes are needed when connecting to the DDAG share, but changes needed when connecting in PDX.
