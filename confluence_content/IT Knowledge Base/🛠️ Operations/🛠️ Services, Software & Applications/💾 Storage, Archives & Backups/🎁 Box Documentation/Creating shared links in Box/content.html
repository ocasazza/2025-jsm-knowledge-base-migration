“Shared links enable you to quickly copy and share hyperlinks to content stored in Box with your coworkers and friends – both inside and outside the company. Sending someone a shared link to a file or folder is a way to work together with that person around shared content. Customizable permission levels, expiration dates, and optional password-protection make Box shared links a secure, simple way to share important content.” (Ref https://support.box.com/hc/en-us/articles/360043697094-Creating-Shared-Links )

  1. As of , we have changed our Box settings to set a default expiration for shared links created after the change.

    1. If you hover over the red icon or click Link Settings, you can see the expiration date

    2. At any time during or after the creation of the shared link, you can click Link Settings to change or even remove the shared link.

    3. Once a shared link expires, the URL associated with that shared link will no longer work. If you create a new shared link on the same item, it will have a different URL.

  2. More information about shared links can be found on https://support.box.com/hc/en-us/articles/360043697554-Configuring-Individual-Share”d-Link-Settings
