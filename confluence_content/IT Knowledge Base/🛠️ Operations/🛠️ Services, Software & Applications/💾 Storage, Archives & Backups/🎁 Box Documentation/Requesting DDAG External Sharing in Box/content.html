Procedure for sharing DDAG project materials with people internal and external to Schrödinger.

Purpose

This procedure will ensure that requests to share DDAG materials using Box  –  for people both internal and external to Schrödinger – are verified as correct and approved by senior management.

Background

We run many projects within DDAG, with many external parties participating in various projects.  All projects are highly confidential and data must not be shared with an unintended or unauthorized person or company. In order to properly put "guard rails" in place, we have implemented a basic approval workflow to ensure that requests to share data using Box are authorized.

Be aware that when a folder is shared in Box, ALL content within that folder is shared without exception, including all individual files within the subfolders.


Permission Levels

Default Box sharing levels have been changed for this organization and " Editor " roles can not invite other collaborators into the folder they're added to as an Editor. If you would like someone to be able to invite more collaborators, please make them " Co-owner ". Thus our organization's permissions are slightly different from ones described in https://support.box.com/hc/en-us/articles/360044196413-Understanding-Collaborator-Permission-Levels


  1. We only have Co-owner, Editor, Viewer-Uploader, Viewer and Previewer Roles.
  2. Opposing the default settings in Box, Editor roles do not give the user permission to add someone new to the folder.
  3. This causes confusion since the default description of "Editor" role in dialog boxes still indicates the user would be able to "upload, download, preview, share and edit"
  4. Also, please not that link sharing only works for people who are already invited to the folder. If you share the link with someone new, they wouldn't be able to see anything and receive an error.

Scope

This procedure applies to the following folders in Box:

  • DDAG/Internal Projects
  • DDAG/NewCo Projects

Procedure

In order to request that a folder within a DDAG project be created please reach out to who is our Box Content Expert.
