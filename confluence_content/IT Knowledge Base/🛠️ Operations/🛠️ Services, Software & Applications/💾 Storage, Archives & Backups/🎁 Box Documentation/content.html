  3. Work with IT to get the accounts needed for all internal collaborators, establish what kinds of access level each user needs, and also the desired folder structures of your project, if applicable.


Individual File Sharing

  • When a Box account is provisioned, a home folder named “firstName.lastName” will be created for you. Make sure you can see that folder. Feel free to create as many subfolders as you deem necessary.
  • If you want to invite collaborators to the folder, please click “ Share ” and start typing their names for internal collaborators, giving them the appropriate permission levels. For external collaborators, paste their email address(es).
  • We suggest that you add Schrodinger employees as Editors and external collaborators as Viewer-Uploader .

  • All collaborators must have a Box account or be willing to sign up for one - it’s free .

Receiving Files


  • Box has a feature that allows external users to upload files in a Box folder we provide them without needing a Box account.
  • Read more about File Request here . You will have to be a Co-owner of the folder where you collect files into.
  • Click on the three dots on the top right side and select File Request . Customize the settings as needed, you can also see what the preview would look like. The sender doesn’t need a Box account.

Frequently Asked Questions (FAQ)


  1. I can’t see Box in my Okta Dashboard.
    1. Please reach out to IT to provision you with the Box App on Okta.
  2. I can’t see my personal folder and can’t upload anything.
    1. This might happen if you registered for Box prior to Schrödinger's implementation.
    2. Please accept the Box email invitation to be a managed user under Schrödinger's Box account.
    3. Please reach out to IT to provision you with a personal folder in your name.
  3. I can’t share folders with external parties / I see an error that says "Collaboration is restricted within Schrödinger."
    1. This is the default setting for all accounts. Please reach out to IT and ask them to lift the restriction on your account.
  4. I don’t want my collaborators to see who has access to the folder.
    1. If you are the Co-Owner, go to the folder settings by clicking on the 3 dots after selecting the folder and change the setting under the "Privacy" section.
    2. Check out other settings and adjust as needed.
