How to Mount SMB exports on Windows?

Please follow these steps to mount a regional SMB export on a regional Windows system.

1. Open windows explorer and right click on "This PC" on the left on the window manager):

2. Select "Map network drive...":

3. Set the "Drive" to the highest letter available, and set the "Folder", which is the path of the SMB export (in the form "\\server\share-name")

4. Ensure "Reconnect at sign-in" is checked and then click finish and the export should open in a new explorer window

6. If you want to disconnect the mount, right click on the drive letter, "Z:" in this case, and select "Disconnect"
