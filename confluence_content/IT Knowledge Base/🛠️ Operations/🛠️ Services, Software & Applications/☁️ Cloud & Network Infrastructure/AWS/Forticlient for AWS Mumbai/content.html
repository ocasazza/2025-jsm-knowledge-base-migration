The following instructions detail how to connect to the VPN in AWS Mumbai for the people under the DESIS / Schrodinger collaboration. This setup replaces the need for the "PDX VPN". Please note that if you were previously using FortiClient for VPN, you should update to the latest version in step 2 below.


Step 1

Ensure that you have Okta Verify Installed on your phone and connected to your Okta account.

  • Okta Verify for iOS: https://apps.apple.com/us/app/okta-verify/id490179405
  • Okta Verify for Android: https://play.google.com/store/apps/details?id=com.okta.android.auth&hl=en_US&gl=US

You can connect your Okta Verify application to your Okta account on your Okta Profile page.


Step 2

Download and install the  Forticlient VPN package on this page: https://www.forticlient.com/downloads
**Note the VPN client only works on MacOS and Windows. If a Linux VPN client is needed, please file a sysmgr ticket.


Step 3

Configure the client as follows. Simply substitute your own Okta login name (in email address form) for the username.  The shared secret is meteors80citric and the Local ID is Schrodinger .




Step 4

Connect. You will receive a push notification from the Okta verify application that you must accept. This will be required every time in order to log in.
