Overview

Zoom and Google Meet are our standard applications for meetings. However, in certain use cases, Microsoft Teams may need to be used.

Attending Microsoft Teams meetings

  1. You should normally not need to be signed into your Microsoft Teams client with a Schrödinger account in order to just attend a meeting hosted on Microsoft Teams, but there could be exceptions. For example, if someone at Schrödinger is hosting the meeting, and has specified that attendees must be signed in to join the meeting.

Hosting Microsoft Teams meetings

  1. Accounts for hosting meetings with Microsoft Teams are not provisioned by default. If you have a use case that requires you to be able to host Microsoft Teams meetings, please open a SYSMGR ticket .

  2. If you are granted a license for hosting meetings with Microsoft Teams, it will be as part of a subscription for Office 365.  Instructions for installing and activating the applications for Office 365 can be found here .

  3. In order to be able to schedule Microsoft Teams meetings from your Schrödinger Google Calendar, you must install the Microsoft Teams add-on for Google Workspace:

    https://workspace.google.com/marketplace/app/microsoft_teams_meeting/961264221134 and choose Individual Install.
