Step-by-step guide

Add the steps involved:

  1. Once you have been assigned DocuSign, please look for an activation email from DocuSign.  You'll need to sign into Okta during the activation, if you aren't already signed in.
  2. Once your account is activated, you can launch DocuSign from your Okta dashboard .
  3. We have several DocuSign accounts (Standard, CFR Part 11, QES). If you have a user account in more than one DocuSign account, you may see a chooser screen where you will need to pick the account you wish to use. If you are using the Standard or CFR Part 11 account, you'd pick the one that lists (NA3).  If you are using the QES account, you'd pick the one that lists QES (NA4).

  4. Once you are logged in, if you need to verify which account you are using, you can click on your initials in the upper right:
  5. You can also use the Switch Account button, if you have it available, to switch between different accounts. For example, you'd use it to switch between the Standard and CFR Part 11 DocuSign as they are both on NA3.  To switch to the QES NA4 account, you'd need to log out and then pick that from the chooser screen.
  6. For more information, see these DocuSign support articles:
    1. Switch between accounts on different sites
    2. To switch between accounts on the same site
    3. Switch Accounts and Set a Default Account
