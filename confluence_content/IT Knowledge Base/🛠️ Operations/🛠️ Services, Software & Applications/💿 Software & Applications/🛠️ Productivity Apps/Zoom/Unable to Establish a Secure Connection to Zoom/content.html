Problem

At times the Zoom client for Mac will give error Unable to establish secure connection to Zoom and will show a certificate error.

Solution

Below is an example screenshot of the issue:

To help resolve this issue please attempt the following:

  • The simplest solutions to attempt is to reboot the computer and launch the Zoom client again.

  • If rebooting doesn't work uninstall and reinstall the Zoom client from the computer

    • Steps to uninstall an application can be found in this Apple Support Article

    • To install Zoom again you can use this link for download and instructions
