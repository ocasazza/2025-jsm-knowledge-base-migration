This installation guide applies to anyone Tableau user with a "Creator" role. Those with "Explorer" & "Viewer" roles will not have access to Tableau Desktop.

Installation Steps:

1.) Access Tableau Online via Okta tile.

2.) From the home page, click "Download Tableau Desktop".

3.) You will be prompted to create an account. Please do so with your Schrodinger email.



4.) Select the latest version of Tableau to download.

5.) Download for your respective Operating System and follow the installation instructions.

6.) Once installed open the application and find the setting tab for "Server".



7.) Select Tableau Cloud from the drop-down.



7.) You will be prompted with a login screen. Please enter you Schrodinger email.

8.) You will be redirected to an Okta login screen. Continue authenticating with your Okta credentials.

9.) Following completion of these steps, you will be authenticated with our Tableau Cloud instance. Please verify you can access data sources from the cloud by selecting "Tableau Server" under "Search for Data".
