Problem

You see a WS1 window open when you login into Windows:

Solution

  1. Please first find a time when you can leave that window open for a while to see if it completes and changes the status.  This might work, but could take some time, so it's best to try leaving it overnight if possible.

  2. If you find that this doesn't help and it keeps opening at each login, again leave it open for a while, to see if it gives an indication of any particular application that is causing an issue (it might be marked with a red exclamation point.)  If so, then if you have an existing Jira ticket open for this problem, attach a screenshot.  Then uninstall that application from Windows . WS1 will try to re-install it, likely after a reboot.
