In addition to the Okta Verify mobile app, you can use the built-in fingerprint reader on your devices or hardware security keys as a second factor.

It is strongly recommended to configure a factor as described here in addition to the Okta Verify app. Otherwise, if you lose access to your current phone you will not be able to complete the Okta login process.

Configuring multiple additional factors is supported.

Prerequisites

  • Using biometrics is supported on Chrome on macOS and Windows and Firefox on Windows

  • To use the built-in fingerprint sensor on device with Okta, first make sure your fingerprint is enrolled with Touch ID / Windows Hello , then follow the steps below to add the device to Okta

    • If you have not yet added your fingerprint on macOS, go to System Preferences : Touch ID and add a fingerprint there.

Adding a new authenticator to Okta

  1. On the device where you wish to use your fingerprint, visit your Okta settings page: https://schrodinger.okta.com/user/settings/account

  2. If you see an " Edit Profile " button in the top right corner, click it

  3. Scroll down to the " Extra Verification " section

  4. Click " Set Up " (or " Set up another " if you have already enrolled a device)

  5. Under "Additional optional factors" click " Set up " (or " Set up another ")

  6. On the following page, click " Enroll "

  7. Choose " This device " to enable Touch ID/Fingerprint Authentication.
    Or choose "USB security key" to set up a U2F Key

  8. The following pop-up will show if you are on a Mac, now tap your fingerprint sensor on your laptop. (You have to have your fingerprint enrolled in Touch ID prior to this step).

These pop ups might be slightly different depending on your browser versions.

If you have any issues enrolling your fingerprint or another U2F Security key please start a new thread in the IT Support Hotline or create a ticket in IT Help/Freshservice.
