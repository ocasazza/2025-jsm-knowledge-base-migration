If you are facing issues where chrome is creating multiple chrome profile every time you launch chrome app

Option 1:

Step-by-step guide

If you see chrome creating different profile, check "Keep local browsing data (bookmarks, passwords, history, etc.)" This will create the new profile and keep your bookmarks, passwords, history, etc. and will stop chrome from creating multiple profiles.

Option 2:

If Option 1 does not work.

WARNING

Deleting the "Google" folder will delete all chrome app data including chrome bookmarks, history and etc.

Step-by-step guide

For Mac: use MAC UI

  1. Make sure to backup your bookmarks, following steps will delete all bookmarks.

    • How to backup bookmarks?

      • In chrome search bar type or click chrome://bookmarks/

      • Click on the 3 dots

      • Click on "Export bookmarks"

      • Save

  2. Quite chrome and locate and delete the "Google" Data Folder

    • Go to "~/Library/Application support/"

    • Locate "Google" folder

    • Move to trash

  3. Restart Chrome

  4. This should solve your issue but if not:

    • Repeat step 2

    • restart uninstall and reinstall chrome

    • Open chrome app

For Windows:

  1. Make sure to backup your bookmarks, following steps will delete all bookmarks.

  2. Quite chrome and locate and delete the "Google" Data Folder

    • Open File Explorer

    • Go to "%USERPROFILE%\AppData\Local\"

    • Locate "Google" folder

    • Right click and delete

  3. Restart Chrome

  4. This should solve your issue but if not:

    • Repeat step 2

    • restart uninstall and reinstall chrome

    • Open chrome app
