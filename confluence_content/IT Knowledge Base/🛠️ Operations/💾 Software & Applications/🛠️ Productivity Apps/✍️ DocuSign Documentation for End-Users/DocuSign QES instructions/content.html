Background to QES signatures

Qualified Electronic Signatures (QES) are often used for high-value, regulated, or cross-border agreements. Under European Union eIDAS law, it’s the only form of digital signature that is considered equivalent to a handwritten signature. A QES requires face-to-face identification or the equivalent.

We are currently using DocuSign QES signatures to enable the use of electronic signatures on employee contracts during onboarding in Europe.

How to use QES

In order to enable a physical person to execute a QES on a document, it is required by regulation that the identity of this person has been previously verified in face to face.

What is ID Check Remote for QES?

  • ID Check Remote for QES is a service of DocuSign that enables individuals to execute a QES on one or several electronic documents

  • The signer is initially identified via video-chat when using the service for the first time. This identification process takes 3-5 minutes

  • At the end of the video-identification process, the signer creates a ‘wallet account’ by choosing a password. This password will be used for any further execution of QES by that same signer.

  • The video-identification is renewed every 2 years.
