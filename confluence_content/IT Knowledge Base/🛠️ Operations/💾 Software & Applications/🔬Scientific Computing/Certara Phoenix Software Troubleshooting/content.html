The Certara Phoenix Software Suite includes various modules, such as WinNonlin, Different users have different modules licensed.

We've included some general troubleshooting tips below.

Installation

  1. Wait until new license codes have been issued, then uninstall the existing software and reboot , before trying to install the new version.

  2. Use 7-Zip or a third-party unzipping program to unzip the installation file. The Windows built-in unzipping sometimes may not unpack the Certara software installation .zip file correctly.

  3. When trying to copy and paste the license code into the software's license key panel, at least one user encountered a problem pasting the license code using the mouse right-click.Â  The user had to use the Windows keyboard shortcut (Ctrl-V) to paste the license code instead.

Licensing

  1. Occasionally Certera Software will stop working on all networks other than the one it was activated on initially. This will make it only usable from that external IP. This problem typically coincides with license reactivation.
    The fix is for Certera to send an new activation code to the user which they will need to use for activation on the network they are having the issue with.
    If you run into this issue, email the following information to certara.licensing@certara.com :

    1. The "lservrc" license file from the folder C:\Program Files (x86)\Certara\Phoenix\application\Services\Licensing

    2. Your Lock Code: Run Phoenix as administrator - Edit - Preferences - Licensing - License Acquisition - Offline tab - Copy Lock Code button.
