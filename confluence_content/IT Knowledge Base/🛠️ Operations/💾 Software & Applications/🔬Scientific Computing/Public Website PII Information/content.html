Background

Users and employees may need to register accounts on https://schrodinger.com for either downloads of Schrödinger software or support services. Personally Identifiable Information (PII) is collected as a part of the registration process. The controls, encryption, workflow, and privacy policy for these accounts whether employee, contractor, or customer are the same.

  1. User registers for a schrodinger.com over encrypted HTTPS using TLS

  2. User account details are stored directly into Netsuite

    1. User’s password is encrypted using PKDBF2

    2. Other fields are stored in plain text

    3. Other data captured:

      1. First Name

      2. Last Name

      3. Email Address

      4. Mailing Address

    5. All user data in Netsuite is encrypted at rest using AES-128 encryption at a minimum.

  4. User data is then synced from Netsuite to the schrodinger.com user database

    1. Transmission from Netsuite to schrodinger.com is encrypted using TLS

    2. Schrodinger.com website database is encrypted at rest using AES-256 encryption

Privacy & Account Deletion

The Schrödinger Privacy Policy lays out the privacy controls that apply to the use of the PII collected during the account creation process. For all cases, whether customer, contractor, or employee, account information is only purged or altered by request to remove@schrodinger.com .

It should be noted that departing employees are not proactively removed from the system. If these departing employees wish their information to be purged, they must follow the same procedure outlined above.
