Installation

The software is installed in NYC in /nfs/dbs/cambridge .

We only have a license for NYC.

Installers are located in /nfs/dbs/cambridge/Installers/latest

Older versions of this do exist in the filesystem but with expired licenses these are of limited value.

CSD Software Available

  • ConQuest - The Interface for the CSD System

  • Mercury - Crystal Structure Visualisation and Exploration Made Easy

  • Mogul - A Knowledge Base of Molecular Geometry

  • VISTA - Statistical Analysis and display of Geometric and Other Data

  • IsoStar - A Knowledge Base of Intermolecular Interactions

  • PreQuest - Creation of In-House Databases in CSD format

  • Find structures with WebCSD

    • Cross-platform web-based structure retrieval to support your

  • Search with ConQuest

    • Answer very specific structure questions on your desktop using

    • Quickly identify structures based on a wide range of properties

    • Use interaction and structure-property knowledge to drive design

    • Represent molecular conformations, crystal structures,

      graphics and videos

  • 3D print your molecules and structures

      information on preferred molecular geometries
