Prerequisites

  1. You need to be plugged into the office wired network, or connected to the VPN

Adding a printer macOS:

  1. Open System Preferences

  2. Select Printers & Scanners

  3. Click: " +"

    1. Select IP on the top row

    2. In Address type in the printers address: PrinterName.schrodinger.com (ex. nyc-23-printer.schrodinger.com )

    3. Confirm by clicking Add

Adding a printer Windows 10:

  1. Open Settings app from the Start Menu

  2. Click Devices

  3. Click Printers & Scanners from the left pane.

  4. Click the " +" to Add a Printer or Scanner

    1. select printer you want add and add printer

    2. If printer does not show up click The printer that I want isn't listed

    3. On the next screen select Add a printer using TCP/IP address or hostname

    4. On the next screen enter the printers address as PrinterName.schrodinger.com and click next (ex: nyc-21-printer.schrodinger.com)

    5. On the following screen click next

    6. On the sharing screen select Do not Share

    7. You have successfully added a printer

Removing a printer macOS:

  1. Open System Preferences

  2. Select Printers & Scanners

  3. Select the printer you want to remove from the left pane

    1. Click: "-"

    2. Confirm you want to delete the printer

Removing a printer Windows 10:

  1. Open Settings app from the Start Menu

  2. Select Devices

  3. Select Printers & Scanners from the left pane.

  4. Select the printer you want to remove from the list

    1. Click remove device

Print jobs not completing:

  1. First make sure that you are connected to Appgate

  2. If connected to Appgate make sure you are using the full FQDN (fully quantified domain name) - <printer name>.schrodinger.com

  3. If all the above is working as expected and you can not still not print you made need to install the printer drivers. These are specific for the model and brand of printer

    1. i.e. Brother (Printer pdx-printer-admin.schrodinger.com) - Driver Page (Pro Tip for macOS - choose macOS 10.15)
