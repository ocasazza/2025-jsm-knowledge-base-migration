Zebra Notebook Setup

  1. Turn on Device by pressing the silver power button on the top left until it vibrates.

  2. You’ll see a couple of splash screens and be presented with a screen that says “Hi There” Tap the Yellow Start Button

  3. Connect to Schrodinger WiFi (Password: M0l3cul3s!)

  4. Do not copy apps and data

  5. There will be a couple of loading screens and you will be asked to sign into google.
    Sign in using you schrodinger email and password

  6. Verify Using 2 factor and tap agree to add this account

  7. Turn off Use Location, Allow Scanning and Sund usage and diagnostic data and accept

  8. Setup Passcode

  9. If you want to use google assistant agree if not skip

  10. Skip Google Pay and tap no thanks on anything else.
    You should now see a desktop!

  11. Next we need to download appgate so go to the google play store and search for appgate

  12. Tap install to download the appgate SDP Client and tap open

  13. Tap approve and then tap use profile link

  14. Now that we are here we’ll need to get you that link so open chrome and sign into schrodinger.okta.com

  15. Once in tap the confluence tab

  16. Tap Search in the top right corner and type in “Appgate Installation” and select the first result

  17. Scroll Down through the document to Step 2 and tap on the blue link that says “SDGR Profile Link”

  18. That will then open up appgate to the Create Profile window.
    Tap
    “Submit”
    “Connect to Appgate SDP”
    “Sign In With Provider”

  19. You will then need to sign into okta for verification and that completes the setup


You should now be able to access the ELN through chrome with no problem
