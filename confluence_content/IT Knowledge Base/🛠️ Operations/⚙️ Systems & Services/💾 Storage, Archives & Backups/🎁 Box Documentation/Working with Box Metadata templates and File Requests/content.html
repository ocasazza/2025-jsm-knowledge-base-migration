Specific Notes

Updating a metadata template

  1. You must have been granted the Box Co-Admin privilege for: Create and edit metadata templates for your company
  2. Go to https://schrodinger.app.box.com/master/metadata/templates
  3. Carefully examine the list of templates, and click on the one you wish to edit. Be very sure you select the correct one, especially if there are several which are similarly-named.
  4. Once you are sure you are looking at the correct template, click the small > next to the item you wish to edit, to expand the section for that item. There you can add, edit, or delete values.
  5. For more detailed instructions, see https://support.box.com/hc/en-us/articles/360044194033-Customizing-Metadata-Templates .

Updating a File Request form after editing associated metadata template

  1. If you have a File Request where the drop-down values are populated from a metadata template, and you make changes to that metadata template, those changes won't automatically show up on the File Request form.  You'll need to edit the File Request form, delete and re-add the associated field, make any other re-adjustments (mark as Required again, for example), then Save.
  2. For example:
    1. Go to the edit view of your File Request form. Note the field's current settings (if it's set as Required, for example), then delete the field (don't click Save yet):



    2. Re-add the field.
      1. Move your mouse between two fields until you see the blue-line and + indicating you can add a field.



      2. Click the +, then click Add a field +.




      3. Choose File Metadata.



      4. The correct template should already be selected, but if not, select it.



      5. Once the correct template is selected, select the correct field.




      6. Click Add Metadata Fields.



      7. Make any adjustments to the re-added field, like selecting if it should be Required.



    3. Click Save (not Share, though that is highlighted in the screenshot)


  3. Go to your File Request form in view mode and refresh it. Verify your new drop-down value is visible. If it isn't, wait a few minutes and try repeating the procedure above.
