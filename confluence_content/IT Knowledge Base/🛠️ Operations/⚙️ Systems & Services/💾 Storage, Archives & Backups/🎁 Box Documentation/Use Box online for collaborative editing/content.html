INLINE

Box enables and supports collaborative editing and versioning of documents both through Microsoft Office Online and Google Apps.


Notes and information

TL;DR

Box put together a short 3 minute video providing an overview and demo of these features.


Caveats

There are a few constraints and limitations to keep in mind here.

  • When a document is actively open for editing either through Office Online or Google Apps it cannot be opened at the same time in the other. How it behaves depends on:
    • If opened first with Google Apps, the Microsoft options will disappear from the menu
    • If opened first with Microsoft Office, Google will still be an option but it will not work and will generate an error
  • Box documents can be edited using a local copy of Microsoft Office instead of online. When doing this it is highly recommended that you lock the document to prevent others making simultaneous changes while you're working offline. If you don't lock the file and someone else makes changes online uploading the offline edits will produce a new file.
  • A document cannot be moved or renamed while it is being actively edited.


For someone else to edit a document collaboratively it must first be shared with them. See related documents or .

Step-by-step guide

All of these steps presume you've logged into Box and navigated to the document you'd like to edit or created a new one within an appropriate folder

Open a document

  1. Click the file name to open a preview of the document

  2. Find and click the Open button in the upper-right hand corner to accept the default editor

    Screenshot...


  3. If you'd like to open with an alternative click the drop-down beside the Open button

    Screenshot...


Save a document

If you are using one of the online editors there is no need to save the document. It will be saved automatically and new version(s) created.

Examine the version history

  1. Click the file name to open a preview of the document

  2. Click the ' ... ' button to bring up the menu and select Version History


  3. The versions should appear in a list along the right-hand side and selecting a particular version should display a preview of the document as it existed at that time.
