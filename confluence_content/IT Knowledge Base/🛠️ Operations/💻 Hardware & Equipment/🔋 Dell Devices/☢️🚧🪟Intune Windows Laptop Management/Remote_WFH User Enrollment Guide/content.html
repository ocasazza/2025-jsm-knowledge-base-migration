Prerequisites

  • Active directory credentials

  • Windows device (minimum Windows 10 v1909)

  • Internet connection with outbound HTTPS (port 443)

Enrollment Process

  1. Navigate to $ENROLLMENT_PORTAL_URL (typically https://aka.ms/intuneportal )

  2. Authenticate using your domain credentials ( username@$COMPANY_DOMAIN )

  3. Complete MFA verification if it is configured

  4. Select "Devices" > "Enroll device"

  5. Download and execute the Intune enrollment application

  6. Choose "Set up for work or school" when prompted

  7. Enter your corporate email address

  8. Complete authentication and follow the system prompts

  9. Verify enrollment status in Settings > Accounts > Access work or school

Enrollment Troubleshooting

  • Authentication failure : Verify your credentials and network connectivity. If the issue persists, see [Self-Service Password Reset|#password-reset].

  • Error 80180018 : Device is already enrolled. Contact $IT_SUPPORT_EMAIL for assistance.

  • Timeout during enrollment : Ensure outbound connectivity to *.microsoft.com endpoints.

  • Error 80004003 : Restart your device and retry the enrollment process.
