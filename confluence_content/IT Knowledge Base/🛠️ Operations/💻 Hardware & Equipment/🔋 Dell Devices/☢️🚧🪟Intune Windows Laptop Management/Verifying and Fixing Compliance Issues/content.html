Compliance Status Verification

  1. Open the Company Portal application

  2. Navigate to the Devices section

  3. Select your current device

  4. Review the compliance status indicator:

    • Green checkmark: Compliant

    • Yellow warning: Remediation required

    • Red X: Critical compliance failure

Common Compliance Issues

Operating System Version

Detection:

  • Navigate to Settings > System > About

  • Verify that the Windows version meets minimum requirements (Windows 10 v1909+)

Remediation:

  • Open Settings > Update & Security > Windows Update

  • Select "Check for updates"

  • Install all available feature updates

  • Restart your device when prompted

  • Force a compliance check with Start-ScheduledTask -TaskName "\Microsoft\Windows\Intune\PushLaunch" in PowerShell

Security Configuration

Windows Defender Status

  • Open Settings > Update & Security > Windows Security

  • Verify that all protection modules show "On" status

  • Run a full system scan if AV definitions are outdated

BitLocker Encryption

  • Open Control Panel > BitLocker Drive Encryption

  • Verify that the system drive shows "BitLocker On"

  • If not enabled:

    1. Select "Turn on BitLocker"

    2. Choose a backup method for the recovery key

    3. Select the encryption method (XTS-AES 256 recommended)

    4. Choose the "Encrypt entire drive" option

    5. Wait for the encryption to complete (this is a background process)

Windows Firewall

  • Open Settings > Update & Security > Windows Security > Firewall & network protection

  • Verify that all network profiles show "On" status

  • If disabled, click each profile and toggle to "On"

Update Compliance

Windows Updates

  • Open Settings > Update & Security > Windows Update

  • Install all pending updates, including optional updates

  • Verify that the last successful update is within the compliance window (typically 14 days)

Microsoft 365 Apps

  • Open any Office application > Account > Office Updates

  • Select "Update Now" if available

  • Verify that the build number meets minimum requirements

Post-Remediation Verification

After addressing compliance issues:

  1. Open the Company Portal

  2. Select Devices > Current device

  3. Click "Check Status"

  4. Wait for the compliance scan to complete (approximately 5 minutes)

  5. If compliance issues persist, execute $env:ProgramFiles\Microsoft Intune Management Extension\AgentExecutor.exe -enforceRemediation in an elevated PowerShell

  6. For ongoing issues, contact $IT_SUPPORT_EMAIL with your device name and logged-in username
