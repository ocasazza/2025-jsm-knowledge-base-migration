Accessing the Company Portal

  1. Launch the Company Portal via the Start Menu or search for "Company Portal"

  2. Authenticate with your domain credentials if prompted

  3. Ensure the connection status shows "Connected" in the portal header

Software Deployment

  1. Navigate to the "Apps" section in the left navigation pane

  2. Filter available applications by category or use the search function

  3. Select the required application

  4. Click "Install" to initiate the deployment

  5. Monitor the installation progress in the notification area

  6. Verify that the application appears in the Start Menu after installation is complete

Installation Troubleshooting

  • Error 0x87D1041C : Insufficient disk space. Free up a minimum of 1GB and retry.

  • Error 0x80070005 : Elevation required. Right-click on the Company Portal and select "Run as administrator".

  • Pending status for over 30 minutes : Force a policy refresh with gpupdate /force in an elevated Command Prompt.

  • "Device not compliant" message : Refer to the [Compliance Verification|#compliance-issues] section.
