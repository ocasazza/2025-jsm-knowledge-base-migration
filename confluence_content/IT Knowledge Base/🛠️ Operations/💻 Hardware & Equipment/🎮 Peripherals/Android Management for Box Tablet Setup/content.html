Create a child OU under the main parent OU.

Creating Settings for Advanced Mobile Management

In order to manage and have more control of devices, you will need to make sure Advanced Management is applied in the settings for that OU. Article on Advanced Management for reference: https://apps.google.com/supportwidget/articlehome?hl=en&article_url=https%3A%2F%2Fsupport.google.com%2Fa%2Fanswer%2F7396025%3Fhl%3Den&assistant_event=welcome&assistant_id=mdmbot&product_context=7396025&product_name=UnuFlow&trigger_context=a

  • In GAdmin, go to Devices > Mobile and Endpoints >Â  Settings > Universal > General > select the OU you want to apply the Advanced settings to > Override

Android Settings

The following settings will be in Devices > Mobile and endpoints > Settings > Android .

Make sure to select the correct OU you want to apply the settings to before applying a change

  • Work Profile: Enable Work profile creation

  • Users and Accounts:

  1. Accounts: Toggle off Allow User to add or remove accounts

  2. Add users: Toggle off Allow user profiles

  • Device Features

  1. Factory Reset: Toggle off Allow users to factory reset a device

  2. Administrator restriction pin: Toggle on Set administrator PIN remotely. Admin pin is in LastPass under, GAdmin Android Restriction Pin

  • Apps and data sharing: Available apps > toggle Only allowed apps

Application Settings

You can apply domain, allowed and blocked apps for specific OU's, once the user add's their schrodinger account via the Google Play store on the tablet

  • Apps > Google Workspace Marketplace apps

Boox Tablet Setup

  • Turn on the tablet and run through all the general setup prompts (Power: Auto Sleep: 5 Mins; Auto Power-Off: Never; Navigation Bar, not Gestures, then default settings for the rest. While setting up the Wifi, select "Don't allow Onyx Boox to record audio") and you will be presented with the homepage, as follows:

Note: The serial number on the tablet will not show up as assigned, due to the devices being imported having a unique identity that is made up of an IMEI number, serial number and model number. I was able to find this number by importing the device with it's actual serial number and seeing how Google see's that device. Once I have that the serial number, I will delete the device by going to Devices > Mobile and endpoints > Devices

  • You will need to add the tablet to GAdmin by importing the serial number into the specified spreadsheet noted in the the following article: https://apps.google.com/supportwidget/articlehome?article_url=https%3A%2F%2Fsupport.google.com%2Fa%2Fanswer%2F7129612&assistant_event=welcome&assistant_id=mdmbot&product_context=7129612&product_name=UnuFlow&trigger_context=a

  • Download the csv template (You may need to add the download to Google drive and download as a CSV file from there to properly import it into Gadmin)

  • In GAdmin, go to Devices > Mobile and endpoints > click the +

  • Select the dropdown for Android and import the csv file

  • Once imported, you will need to sign into the tablet in the Google Play store with the user who will be utilizing the tablet and follow the prompts for the setting up a Work Profile.

  • In GAdmin, if you go to Devices > Mobile and endpoints and search for the recently enrolled device, it will show as a company owned device.

Tablet Home Screen and Settings Setup

  • We can add apps from the Google Play store that can be installed on the tablet once the user signs into the tablet and they will pre-install on the home screen. We cannot block or restrict the Boox local applications since they are not built in apps for Boox.

  • To turn off Boox Sync, you will navigate to the Notes app on the home screen > Tap the hamburger in the top-right corner > Sync Settings and toggle off note to Onyx

  • To clean up the excess local apps, you will need to move them into a folder and place on the 2nd page. The only apps that should be on the home screen's task bar should be the downloads, Google Play store, settings and scan application.

That's it! You can pass the tablet off to the user and it can be managed through Google Workspace .
