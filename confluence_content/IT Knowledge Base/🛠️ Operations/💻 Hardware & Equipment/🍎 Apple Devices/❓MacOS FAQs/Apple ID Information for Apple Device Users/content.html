An Apple ID is needed to download Applications from the App Store on Mac Devices, as well as to utilize other features such as SideCar on iPad.

We do not automatically provision Apple IDs, however they are easy to setup using the instructions below. If you need an Apple ID for any reason, please follow the steps below to create an Apple ID for work.

Steps to Create an Apple ID:

  • Go to go to: https://appleid.apple.com/

  • Click on Create Your Apple ID

  • Fill in the information on the form provided on the website. Ensure you are using your Schrodinger email ( first.lastname@schrodinger.com ) (example of website below)

  • PLEASE USE WORK EMAIL ADDRESS FOR BUSINESS PURPOSES!! == Please Note == at this time w e do NOT provide support for Apple ID's, only Apple can unlock/reset an Apple ID.

    • If the password is forgotten or account locked out, Please Contact AppleCare Support with the below info.

  • A credit card is optional on sign up.

  • If you are prompted to enter an address, we recommend using your office address.
