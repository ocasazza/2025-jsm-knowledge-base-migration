iPad is a great resource for creating notes, reading, document mark-up and doing work quick on the go. Here are some tips and tricks for getting the most out of your iPad.

  • Download Google Drive (from SelfService).

  • Download Box (from SelfService).

  • Import files into the Files app locally to markup documents.

  • To update apps (if getting an update notification) go to Self Service → Find the app needing update → Tap "Reinstall" → Wait for app to complete installation.

  • To rearrange apps on iPad → Tap and hold on the app you want to move until it wiggles → Drag app to desired location.

  • SideCar is a feature that allows you to transform your iPad into a secondary display. See documentation for setup → https://support.apple.com/en-us/HT210380 (If you are signed into a work AppleID on both Mac and iPad, the SideCar Feature can be used to transform your iPad into a secondary display on-the-go.)

  • To create an Apple ID, follow the Apple ID Information article here → https://schrodinger.atlassian.net/wiki/x/bXNLAw

Helpful Info

  • Schrodinger Approved Apps are located in the Self Service application that is installed on the device.

  • If there is an app that you would like to suggest for work, submit a Service Request with IT-Help and include the necessary details such as App Name, a link to the app, and work use justification. We then begin the process by submitting this information for a security review. Once approved, then we deploy the app to the company.

  • Signing into an AppleID is possible, but only necessary to utilize features like SideCar. The Apple ID is highly restricted on an iPad and virtually has no use outside of a few features to sync with your Mac.

  • For exploring more features, view the iPad User Guide → https://support.apple.com/guide/ipad/welcome/ipados


How to Setup iPad

  • Power on Device


  • Choose Language


  • Choose Region


  • Choose a Wi-Fi Network next. If you are in a Schrodinger office, please use the Schrodinger Wi-fi for setup.


  • The Quick Start provides options for setup. In order to setup the iPad properly, select "Set Up Manually." Below is a screen-shot of the Quick Start window and the setup results after tapping Set Up Manually.


  • When the Remote Management window appears, tap Next in the top right corner of the screen and the iPad will begin the first stage of Configuration.


  • If you live in the US, you may setup the Touch-ID or Face-ID options, and must create a 6-digit secure passcode.


  • Enable Location Services


  • Once the steps above are complete, your iPad is setup and ready for you to begin customizing.
