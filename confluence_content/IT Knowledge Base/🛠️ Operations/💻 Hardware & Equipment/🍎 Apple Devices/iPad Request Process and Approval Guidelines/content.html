This documentation outlines the process for requesting an iPad. All iPad requests will undergo approval from the Executive level of your SDGR Organization. To request an iPad, please follow the steps below:

  1. Review the Approved iPad Models Above.

  2. Create a Service Request:

    1. Once the model is decided, create a Service Request in IT Help to initiate the request.

    2. Include the iPad model, yes/no for Apple Pencil, and a case/folio decision for protection.

    3. Include justification for the iPad, use case, etc.

    4. Submit Request.

  3. IT Operations will review the service request and follow up if anything else is needed.

  4. Request is submitted for review to an Executive level to be assed based on budgetary considerations, organizational needs, availability, etc.

  5. Upon approval, IT Operations will order the equipment and follow up on setup procedures - (iPad Setup Guide can also be found here) .
