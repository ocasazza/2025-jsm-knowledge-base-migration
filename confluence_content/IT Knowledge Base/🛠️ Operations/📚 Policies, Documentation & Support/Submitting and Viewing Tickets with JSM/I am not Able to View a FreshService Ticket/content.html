Problem

Someone sends you the URL of a Freshservice ticket, but you're not able to view it.

Solution

If you are not an agent or admin in Freshservice, and not a participant on the ticket, you won't be able to view it using the Agent view URL, which is in the format: https://freshservice.schrodinger.com/a/tickets/$ID. You'll instead need to use the User/requester view URL, which is in the format https://schrodinger.freshservice.com/support/tickets/$ID.

Please note: Certain tickets (like HR, DDAG/STG) have restricted viewing, so switching URL formats will not still not allow you to see the ticket, if you didn't already have the permissions (by being an agent or admin, or a participant on the ticket).

User/requester view URL format

https://schrodinger.freshservice.com/ a /tickets/$ID

https://schrodinger.freshservice.com/ helpdesk /tickets/$ID
