Executive summary

On-call got paged because the admin interface for a PDU in EWR was not reachable.

Background

See System JIRA ac6def98-2140-30c3-8103-4b52b8b31135 ITINFRA-1959 for details

Incident Description

Chronology

Outage was started at 2023-09-05 23:30:45 UTC, ended at 2023-09-06 01:06:38 UTC for a duration of 01:35:53

Corrective Action

See "Introduced Issues" below

Analysis

The only symptom we have is that the PDU stopped responding to LibreNMS polling.  It started responding about 90 minutes later. We don't know if Smart Hands was in the cage before or after it recovered, and they did reseat the network cable on both ends.

Possibilities:

  1. The network cable was loose, and Smart Hands reseated it.  If so, how did it get loose?  When did it get loose?
  2. High power draw from supplied bolt nodes tripped a breaker that is capable of resetting itself.
